Menu as header

  • Make the menu into a big header space
  • Make menu a full-height block
  • Hide logo or links
  • Place logo and links where you want.

Code in the page:

--nav-bg-image
Add an image to the menu

--nav-bg-position
Position the background image

--main-menu-height: var(--space-mega-x2);
Give the menu a height
--main-menu-height: var(--fullheight);
Use this to give it the height of the complete browser window!

--menu-content-justify: flex-start;
--menu-align-items: flex-end;
Horizontally and vertically position the links

--site-logo-height:100%;
--site-logo-position: bottom left;
Position the logo.
