Width of elements

The main section (the page content) has a maximum width of 1200px.

Section elements have a width of 1180px, so the background color of the section shows as a kind of border 10px on left and right side if width of the screen is 1220 pixels or more.

We adjusted the navigation width also, but it could be left fullwidth.
