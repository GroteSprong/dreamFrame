Header

  • Add a header area as a page starter
  • Customize with data-attributes

Code

If you want a header, put the code after the opening body tag, or, if you want the header beneath the menu, after the closing nav tag

Here we have used:
data-window-height="small"
It will scale the height relative to the window height, but you can use all available heights here.

By default the header consists of two equal width parts....

CSS variables to use:

Put an image behind the complete header area:
--header-bg-image: url();
Position the background image
--header-bg-position: center;

Use a background image just for the div area
--header-div-image: url(../img/music.jpg);

Use background colors or background images to layout the blocks...
