Slider

  • In page or as a lightbox
  • One slideshow per page

HTML code

Use the class".gs-slider" to make the div or other block a slider. Add a second class for the name of the slider. "sliderImg" will show the current slide. "forward" the next, "back" the one before. "minmax" will change from inpage slider or fullscreen.
Use your own forward, back and fullscreen images if you want.

JAVASCRIPT code

Open "dreambits.js" in the "js" folder and look for "sliderShow". If you find it, copy it, paste it below the one you copied, and adjust the picture names.
You can add as many pictures you want.

If you would ever like to upgrade Dreambits Framework, it is safer to put all your sliders in a seperate javascript file and link that to the page.

ADJUSTMENTS

If your images are NOT in /img/slider/ , use "imgpath" to point to the right directory. Can be anywhere.

If you want an other animation, use "ani" to set it. Look in the dreambits.css file and search for "animations" to find all the available.
