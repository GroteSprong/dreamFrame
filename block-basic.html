Page Content in Blocks

  • Navigation and footer are full width
  • The page content is inside the MAIN element, in what we call "Blocks"
  • MAIN element is full width
  • Blocks are made up of a SECTION and an ARTICLE
  • SECTION is 100% width of MAIN.
  • ARTICLE is 80.9% width of SECTION.
  • Blocks have a margin between them
  • ARTICLE is centered in the SECTION
  • SECTION can contain multiple ARTICLE elements

All this can be adjusted to your taste, see next block!

These are the properties set up in the top of the CSS file. So you can set a color for the SECTION background, and a color for the ARTICLE font. The article background has been made transparent.

Normally, the "margin" between blocks is actually created by padding on the article. but because the color sceme has switchec -the background color is now on the selection, and the article background is transparent- , the visual margin has disappeared. Therefore, real margins have been added to the section.
