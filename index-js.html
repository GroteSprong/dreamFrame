Fetch!

The Javascript "fetch" function is an addition in javascript, included in every browser. It can be used to load data or pages from your own site, or from other sites (if you have permission)

Fetch works in every browser, is part of the Javascript Open Standard developed bij de Word Wide Web Consortium. As so, it has to work in any device or browser and will be backwards-compatibe, meaning, it will keep working for years and maybe forever.

For more information, check the docs.

CORS

For security, as a default, Javascript can only load data when the HTTP protocol is used (so, on a web server) , and can only load data from the same domain.

This can be a problem when viewing pages locally and making pages.
